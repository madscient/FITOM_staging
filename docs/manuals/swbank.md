# パフォーマンスバンク(SwPatch)マップ

音色本体とは独立して、ベロシティ感度・ビブラート・デチューンなど
の演奏表現を切り替えるための補助バンクです。多くの音色パッチには
あらかじめ標準的な組み合わせが紐づけられているため、通常は
意識する必要はありません。
