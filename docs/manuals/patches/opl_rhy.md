## <a id="cc320"></a>CC#32=0: MSX-AUDIO/OPLL Emulate Rhythm

各パッチは対象楽器(下表「楽器」列)を持ち、同じ楽器に
複数の音色バリエーションが用意されている場合があります。

| Prog | 名称 | 楽器 |
|---|---|---|
| 0 | OPL Bass Drum | BD |
| 1 | OPL Close Hi Hat | HH |
| 2 | OPL Snare Drum | SD |
| 3 | OPL Tom | TOM |
| 4 | OPL Top Cymbal | CYM |
| 5 | OPL Open Hi Hat | HH |
| 6 | OPL Tom (Pitch LFO) | TOM |
| 7 | OPL Snare Drum (Pitch LFO) | SD |
